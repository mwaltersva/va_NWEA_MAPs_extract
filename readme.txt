MAPS testing extract

Installation:

Put maps.html and mapssectiondata.html in /admin/reports/extracts/

Intended to be used with the custom reports bundle from
http://powerdatasolutions.org with dynamic menu generation
turned on.

Some code reused from the custom reports bundle, all
credit for that code goes to it's authors:
http://powerdatasolutions.org

Usage:

Cannot be run from district office.

Run from Start > System Reports > custom reports > extracts > MAPS extract test

Select a course,

submit.
